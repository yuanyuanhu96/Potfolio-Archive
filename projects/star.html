Reflections

In my Prototyping Interaction class I made my own little JavaScript script game. It was really exciting that I got exposed to version control tools, built the file structures, got more practice in HTML, CSS, and tried Javascript.

It was very entertaining at the beginning because I leaned something new. However, after knowing more, I realize there are more I don't know. Now I think I'm becoming more clear about what I should learn and what I have learned. After this project, I started to understand why and how to write clean code - the code is not only written for the computer but also should for us to look back. I would like to learn more about how to write good code, that is scalable.

After all I am very proud of what I have created, and learned :)
